ID e CLASS

ID

O ID funciona como um cpf. assim como so posso ter um numero de cpf, cada elemento so pode ter um valor de ID. O CPF é unico, ou seja, ninguem pode ter um cpf igual ao meu. isso significa que o id tambem é unico, então não é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você so pode ter um valor, no class você pode ter varios valores. tambem diferente do ID, em que não se pode repetir um valor, no class você pode utilizar o mesmo valor para varios elementos.
